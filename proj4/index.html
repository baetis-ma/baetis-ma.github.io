DHT Home Network

This is an esp-idf project with a ESP32S and an HMC5883L I2C magnetometer module (was about a buck on eBay). The webpage updates every 300msec and has a list of measured parameters, an image of the actual orientation of the magnetometer (when the module is upside down the bottom shows), a graph of the x, y and z axis flux, a graph of the extracted spherical coordinates theta (tilt) and phi (level rotation) and a 3D plot of the flux vector. The device is surprisingly accurate after rotating the device through several axis.
