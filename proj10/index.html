LiPo Battery Tool

This is an esp-idf project

The project the following

  • Monitor Cell Voltages of LiPo Battery stack
  • In Charge Mode : charge each battery to 4.1 Volts at preset current while monitoring charge
  • In Discharge Mode : monitor discharge of battery at preset ceurrent while monitoring charge
  • In Discharge to 3.8 Volt Mode : leave battery at 2.8 Volts by discharging or charging
