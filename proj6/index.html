Atmosphere Monitor

This is an esp-idf project with an ESP32S, an SSD1306 128x64 oled display, a BMP280 i2c pressure sensor and a dht22 Temperature and humidity sensor. Readings are taken about once per second and updated to the ssd1306 display.

The only things that made this a not very straight forward project is that I could find and ssd1306 projects that I wanted to copy code from. I ended up working out the i2c display initialization registers and frame array addressing from the device data sheet. I found a 5x7 binary font image table that is included in the ssd1306 include file that is used to convert text to framebuffer writes. A string_display variable is defined that enable the user to control font size and increment the display line pointer.
