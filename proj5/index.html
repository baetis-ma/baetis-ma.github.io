DHT Home Network

This was my first ESP8266 project using nodeMCU and the lua language along withESPlorer or nodemcu-tools to download and run the lua files. DHT Home network collects the data from a number of remote esp8266 connected to DHT-11 temperature and humidity sensors that are broadcasting json data packets to udp sockets. The base station collects the udp packets and conbines them for tsp transmission to the browser.
