MPU6050 Development

This project is a development tool to begin playing around with a MPU6050 monolithic three axis accelerometer and three axis gyroscope. Web controls are available for controlling the register configuration and graphs are available to monitor the six axis measurement data.

Reg 0x19 Sample Rate Divider

Reg 0x1a Configuration dlpf[2:0] = 3 1kz 44hzbw

Reg 0x1b Gyro config sets active axis and range

Reg 0x1c Accel config sets active axis and range

Reg 0x23 FIFO Enable set to 0x78 for 6 axis

Reg 0x24-36 i2c master registers

Reg 0x37-39 Int pins config

Reg 0x3b-40 Accel direct read registers

Reg 0x41-42 temp direct read registers

Reg 0x43-48 gyro direct read registers

Reg 0x49-67 master i2c direct read registers

Reg 0x68 Reset Register

Reg 0x6a User Control

Reg 0x6b-6c Power Register

Reg 0x72-73 FIFO Data Count

Reg 0x74 FIFO Data

Reg 0x19 Sample Rate Divider
